Flexbox = create a flexible layout
Layout = how elements are positioned in the layout

To use flexbox:
1. Create a container around elements
2. Add display: flex; to container
3. Use flexbox features

flex-grow = makes an element grow and take up the remaining space


<div> = takes up entire line by itself = block element 

When a block element is inside a flexbox
    - no longer takes up the entire line
    - only takes up as much space as it needs to


elements inside a flexbox will stretch vertically

align-items controls how elements are displayed vertically
by default its value is stretch

we want the content to stretch all the way down to 100vh
but doing this will create a scroll because by default
body has a default margin of 8px on all sides

flex-direction = which direction the elements are positioned (row by default)

if we want the div to not push the items down as it grows we use the overflow:scroll
scrollbar-width: none; to hide the scrollbar
